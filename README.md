# üìÉ Command Documentation

**`/timezone info`** ‚Äî shows general information about Timezones and how to use it  
**`/timezone setup`** ‚Äî shows info about how to setup Timezones  
**`/timezone check [@user]`** ‚Äî shows you someone's set timezone  
**`/timezone update`** ‚Äî prompts you to update your timezone  
**`/timezone remove`** ‚Äî removes your timezone from your profile

# üöß Setting Up

After adding **Timezones** to your server, follow these steps:

1. Create a new role and call it whatever you want
2. Go to the **Links** tab > **Add requirement** > select **Timezones**
3. You're all done! Now head to your server's Linked Roles and you'll see timezones available

> ### **üí° Tip**
>
> If you own a big Discord server, make sure to let your members know how **Timezones** works!
